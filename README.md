# SkyNow - Weather App

A feature-rich weather application providing real-time weather data, 5-day forecasts, hourly updates, and air quality information with geolocation support.

## Features
- Real-Time Weather Data
- Geolocation Support
- 5-Day Weather Forecast
- Comprehensive Weather Highlights
- Hourly Forecast with Wind Direction

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript (ES6 Modules)
- OpenWeatherMap API
- Geolocation API
- Font Awesome Icons
- Boxicons

## Installation
1. Clone the repository
```bash
   git clone https://github.com/ratneshchipre/Weather-App.git
```
2. Navigate to the project directory
```bash
   cd Weather-App
```
3. Get your free API key from [OpenWeatherMap](https://openweathermap.org/api)
4. Replace the API key in `script.js`:
```javascript
   const API_KEY = 'YOUR_API_KEY_HERE';
```
5. Open `index.html` in your browser
